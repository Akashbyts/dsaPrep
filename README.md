## 💡 Problem-Solving Philosophy

```java
// My approach to every DSA problem
public void solveProblem(Problem p) {

    // Step 1: Understand — Read twice, ask clarifying questions
    String understanding = read(p).carefully().twice();

    // Step 2: Brute Force First — Never skip this
    Solution brute = thinkBruteForce(p);

    // Step 3: Optimize — Identify bottlenecks
    Solution optimized = applyPattern(brute, PATTERNS.bestFit(p));

    // Step 4: Edge Cases — The devil is in the details
    validate(optimized, edgeCases(p));  // nulls, negatives, overflow...

    // Step 5: Code Clean — Write it like it's production
    submit(optimized.withCleanCode().withComments());
}
```
